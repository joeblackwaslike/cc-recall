# Using cc-recall

cc-recall makes past Claude Code sessions discoverable by what was done, asked, and questioned.

**To find a past session:** run `/recall:search <query>` (sidecar-backed) before grepping raw `.jsonl` transcripts.

**Status:** scaffolded (cc-recall-3i1), not yet implemented. See the design spec under `docs/superpowers/specs/`.
